# vite-babylon-gltf-sandbox

## About

Vite vanilla-ts 環境の Babylon.js で SceneLoader を使って
blg モデルをロードするのにちょっとした工夫が必要だったので
その備忘録的なデモ。

## Env

- Babylon.js 5.0.0-alpha.60
- TypeScript
- Vite
- Sass

## Install & Usage

```
# install
yarn install

# launch
yarn dev

# build
yarn build
```
